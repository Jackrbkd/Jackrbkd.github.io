文法和语言的概念和表示

预备知识—形式语言基础

文法的非形式讨论

文法：

​ 文法是对语言结构的定义与描述。即从形式上用于描述和规定语言结构的称为“文法”(或称为“语法”)。

◇ 文法是在形式上对句子结构的定义与描述，不涉及语义问题。

语法规则：

​ 描述句子语法结构的规则。规定用“::=”表示“由……组成”。

<句子>::=<主语><谓语>

<主语>::=<代词>|<名词>

<代词>::=你|我|他

由规则推导句子：

​ 可以按照语法规则来推导或产生句子。

◇ 推导方法：从一个要识别的符号开始推导，即用相应规则的右部来代替规则的左部，每次仅用一条规则去进行推导，直到所有符号被终结符号替代为止。

◇ 最左推导：有若干语法成分同时存在时，我们总是从最左部分进行推导，这称之为最左推导。最右推导同理。

语法树：

​ 语法树用来描述一个句子的语法结构

◇ 语法成分：<>，在形式语言中又称”非终结符“

◇ 单词符号：在形式语言中又称“终结符号”

文法和语言的形式定义

文法：

​ 文法G = ($V_n,V_t,P,Z$),

$V_n$ : 非终结符号集

$V_t$ : 终结符号集 #V = $V_n \cup V_t$称为文法的字汇表

P : 产生式或规则的集合

规则是一个有序对(U,x)，通常写为U::=x或U→x，|U| = 1 |x| $\geq$ 0 , $U\in V_n,x\in V^*$

Z : 开始符号(识别符号) #$Z\in V_n$

直接推导

文法G：v = xUy, w = xuy

其中x、y$\in V_,u \in V_n, u \in V^$

若U::=u$\in P$，则$v\underset{G}{\Rightarrow}w$.

若x = y = $\xi$，有U::=u，则$U\underset{G}{\Rightarrow}u$ #啥意思？感觉多此一举。。。

间接推导

文法G，$U_0,U_1,U_2,…,U_n \in V^+$

if $v \underset{G}{\Rightarrow} U_0 \underset{G}{\Rightarrow} U_1 \underset{G}{\Rightarrow} U_2 \underset{G}{\Rightarrow} … \underset{G}{\Rightarrow} U_n \underset{G}{\Rightarrow} w$,则$v \stackrel{+}{\underset{G}{\Rightarrow}} w$

$v \stackrel{*}{\underset{G}{\Rightarrow}} w$

文法G，v、w$\in V^+$，

if $v \stackrel{+}{\underset{G}{\Rightarrow}} w$，或 v = w，则$v \stackrel{*}{\underset{G}{\Rightarrow}} w$

规范推导

有$xUy \Rightarrow xuy$，if $y \in V_t^*$，则此推导为规范的，记为$x
Uy \nRightarrow xuy$ #ppt中斜杠似乎为竖直杠

直观意义：规范推导 = 最右推导

文法G[Z]

句型：x是句型 <=> Z $\stackrel{}{\Rightarrow}$ x，且$x \in V^$；

句子：x是句子 <=> Z $\stackrel{+}{\Rightarrow}$ x，且x$\in V_t^* $;

语言：L(G[Z]) = {x | x $\in V_t^*$，Z$\stackrel{+}{\Rightarrow}$x}；

等价文法

G和$G^,$是两个不同的文法，若L(G) = L($G^,$),则G和$G^,$称为等价文法

递归文法

递归规则

规则右部有与左部相同的符号，

对于U::=xUy，若x=$\xi$，即U::=Uy，左递归；

若y = $\xi$，即U::=xU，右递归

递归文法

● 文法G，存在U$\in V_n$，

​ if U$\stackrel{+}{\Rightarrow}$…U…，则G为递归文法(自迁入递归)；

​ if U$\stackrel{+}{\Rightarrow}$U…，则G为左递归文法；

​ if U$\stackrel{+}{\Rightarrow}$…U，则G为右递归文法。

● 左递归文法的缺点：

​ 不能用自顶向下的方法来进行语法分析

● 递归文法的优点：

​ 可用有穷条规则，定义无穷语言

句型的短语、简单短语和句柄

短语&简单短语

给定文法G[Z]，w = xuy$\in V^+$，为该文法的句型，

若Z$\stackrel{*}{\Rightarrow}$xUy，且U$\stackrel{+}{\Rightarrow}$u，则u是句型w相对于U的短语；

若Z$\stackrel{*}{\Rightarrow}$xUy，且U$\Rightarrow$u，则u是句型w相对于U的简单短语；

其中U$\in V_n, u\in V^+, x,y\in V^*$

句柄

任一句型的最左简单短语称为该句型的句柄

语法树与二义性文法

推导与语法树

● 语法树：句子结构的图示表示法，它是一种有向图，由结点和有向边组成。

● 结点：符号

​ 根结点：识别符号

​ 中间结点：非终结符

​ 叶结点：终结符或非终结符

● 有向边：表示结点间的派生关系

句型的推导及语法树的生成(自顶向下)

给定G[Z]，句型w：

​ 可建立推导序列，$Z \stackrel{*}{\underset{G}{\Rightarrow}} w$

​ 可建立语法树，以Z为树根结点，每步推导生成语法树的一枝，最终可生成句型的语法树

​ 注意在生成语法树时的二义性问题

子树与短语

● 子树：语法树中的某个结点(子树的根)连同它向下【派生的部分所组成

● 定理：某子树的末端结点按自左向右顺序为句型中的符号串，则该符号串为该句型的相对于该子树根的短语。

​ 那么，我们就可以画出句型的语法树，然后根据子树找短语、简单短语、句柄。

树与推导

● 句型推导过程 <==> 句型语法树的生长过程

由推导构造语法树

由语法树构造推导

● 对句型中最左简单短语(句柄)进行的规约称为规范规约。

​ 通常我们每次都剪掉当前句型的句柄(最左简单短语)，即每次均进行规范规约。

● 规范规约与规范推导互为逆过程

● 通过规范推导或规范规约得到的句型称为规范句型

文法的二义性

● 若对于一个文法的某一句子存在两颗不同的语法树，则该文法是二义性文法，否则是无二义性文法。

● 若一个文法的某句子存在两个不同的规范推导，则该文法是二义性的，否则是无二义性的

● 若一个文法的某规范句型的句柄不唯一，则该文法是二义性的，否则是无二义性的

● 文法的二义性是不可判定的

​ 解决办法：提出限制条件(称为无二义性的充分条件 )，来限制文法

​ 或者不改变二义性文法，而是确定一个满足无二义性充分条件的编译算法

句子的分析

● 任务：给定G[Z]:$S \in V_t^*$，判定是否有$S \in L(G[E])$？

​ 此乃词法分析、语法分析工作，将在三四章介绍

有关文法的实用限制

有害规则

● 如U::=U的规则，会引起二义性

多余规则

● 在推导文法的所有句子中，始终用不到的规则，该规则的左部非终结符不出现在任何句型中

● 在推导句子的过程中，一旦使用了该规则，将推不出任何终结符号串。即该规则中含有推不出任何终结符号的非终结符。

例如给定G[Z]，若其中关于U的规则只有如下一条：

U::=xUy，该规则是多余规则。

● 若某文法中无有害规则或多余规则，则称该文法是压缩过的。

扩充的BNF表示(Backus Normal Form)

● BNF的元符号：< > ::= |

● 扩充的BNF的元符号：< > ::= | { } [ ] ( )

文法和语言分类

● 形式语言：用文法和自动机所描述的没有语义的语言

● 语言定义：L(G[Z]) = ${x|x\in V_t^*,z \stackrel{+}{\Rightarrow} x}$

● 文法定义：乔姆斯基将所有文法都定义为一个四元组：G = ($V_n,V_t,P,Z$)

$V_n$:非终结符号集

$V_t$:终结符号集

P:产生式或规则的集合

Z:开始符号(识别符号) $Z \in V_n$

● 文法和语言分类：0型、1型、2型、3型

​ 差别在于对产生式施加不同的限制

● 0型：

​ P：u::=v，其中$u \in V^+,v \in V^*$

​ 称为短语结构文法，规则的左部和右部都可以是符号串，一个短语可以产生另一个短语

​ L0，可以用图灵机接受

● 1型：

​ P：xUy ::= xuy，其中$U \in V_n,x、y、u \in V^*$

​ 称为上下文敏感或上下文有关，也即只有在x、y这样的上下文中才能把U改写成u

​ L1，可以有一种线性界限自动机接受

● 2型：

​ P：U ::= u，其中$U \in V_n,u \in V^*$

​ 称为上下文无关文法，也即把U改写成u时，不必考虑上下文

​ 1型文法规则中x、y均为$\xi$时即为2型文法

​ L2，可以由下推自动机接受

● 3型：

左线性：

​ P：U ::= t 或 U ::= Wt，其中$U、W \in V_n,t \in V_t$

右线性：

​ P：U ::= t 或 U ::= tW，其中$U、W \in V_n,t \in V_t$

​ 3型文法称为正则文法，它是对2型文法进行进一步限制

​ L3，又称正则语言、正则集合，可以由有穷自动机接受

● 综上，$L3 \subset L2 \subset L1 \subset L0$

​ 0型文法可以产生L0、L1、L2、L3

​ 2型文法只能产生L2、L3

​ 3型文法只能产生L3
